# Purpose

The purpose of this course is to provide the knowledge of bioinformatics with the hands-on practice to the biologists.

# Requirement
- A laptop or computer
- Knowledge of the application of NGS

# Topics

## Basic introduction of the programming environments
- Shell script
- R
- Python

## Pipelines
- nf-core

## Demultiplexing
- bcl2fastq

## Bulk RNA-Seq Analysis
- nfcore-rnaseq
- DGEA with DESeq2

## Learning R for biologists
- Installation of R with Rstudio
- Loading tables or excel files
- Dataframes and how to manipulate it
- Calculate statistics from the tables
- Visualization with ggplot2
- Visualization with plotly
- R markdown report
